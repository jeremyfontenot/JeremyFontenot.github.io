About Me

Hi, I’m Jeremy Fontenot, an IT professional with expertise in Service Desk Support, Systems Administration, and Networking. With a strong background in Active Directory, Microsoft 365, Azure, and network security, I specialize in troubleshooting, system optimization, and IT infrastructure management.

I hold an A.S. in Computer Science - Database Administration and completed the Information Security and Administration Program with a 4.00 GPA. I’m passionate about technology, security, and automation, always exploring new tools and refining my skills to solve complex IT challenges. Whether it’s troubleshooting an issue, designing a network, or optimizing systems, I strive to deliver efficient, practical solutions.

My Enterprise-Grade Home Lab

My home lab is designed to simulate a real-world enterprise IT environment, allowing me to practice system administration, networking, security, and virtualization.

Hardware

  • Hypervisor: VMware ESXi 6 running on a Dell R710
  • Storage: 750GB local storage (HDD/SSD)
  • RAM: 24GB (Optimized for multiple VMs)
  • Networking: TP-Link Archer AX1800 in AP mode
  • Firewall: pfSense for traffic control & security

Virtualization

  • Hypervisor: VMware ESXi hosting multiple virtual machines
  • Windows Server 2022 (AD-SERVER): Primary Domain Controller for authentication
  • Linux VM (RHEL 9): Used for system administration & scripting
  • Web Proxy VM: Reverse Proxy for secure web routing (Apache/Nginx)
  • Monitoring & Security VM: Logs, network monitoring (Zabbix, Wazuh, Graylog)
  • Test VM: Sandbox environment for experimenting with OS & software

Networking & Security

  • pfSense Firewall: Controls WAN/LAN traffic, VLAN segmentation
  • Active Directory: ad.jeremyfontenot.online domain for authentication
  • DNS Server: Windows Server 2022 as the primary DNS resolver
  • VLANs: Configured for separating different network zones
  • VPN: Secure remote access (OpenVPN or WireGuard)
  • Intrusion Detection: Suricata/Snort integration with pfSense

Learning & Experimentation

  • Windows Server Administration: Active Directory, GPO, DNS, DHCP
  • Linux Management: Bash scripting, automation, and security hardening
  • Networking: VLANs, firewall rules, packet analysis (Wireshark, Nmap)
  • Cloud Services: AWS & Azure hybrid infrastructure integration
  • Security Testing: Running penetration testing tools in a safe lab environment
  • Virtualization Experiments: Testing different hypervisors (Proxmox, nested ESXi)
