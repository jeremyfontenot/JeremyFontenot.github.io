Professional Summary

I am an entry-level System Administrator focused on Microsoft-based environments, including Windows Server, Active Directory, Entra ID, Microsoft 365, and Intune. I build and maintain a homelab that mirrors small and mid-sized business infrastructure, giving me practical experience with identity, endpoint management, and network services.

My strengths include troubleshooting authentication and connectivity issues, automating repeatable tasks with PowerShell, and documenting changes in a way that supports incident response and future maintenance. I enjoy turning loosely defined problems into clear runbooks, scripts, and configurations that can be reused by a team.

This portfolio highlights real-world style projects such as Active Directory migration planning, Intune baseline deployment, Microsoft 365 lab design, incident root cause analysis, and user provisioning automation. Each project includes architecture diagrams, scripts, logs, and documentation to demonstrate how I approach system administration work from both a technical and operational perspective.
