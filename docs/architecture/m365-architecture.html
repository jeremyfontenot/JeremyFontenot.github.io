M365 Personal Cloud Lab Architecture

Enterprise-scale M365 lab built on Microsoft 365 Business Premium (tenant: jeremyfontenot.online), demonstrating complete identity, access, Exchange, Teams, Intune, and SharePoint automation using PowerShell and Microsoft Graph exclusively.

Core Architecture

  • Identity: Entra ID (users, groups, roles)
  • Security: Conditional Access (MFA enforcement)
  • Messaging: Exchange Online
  • Collaboration: Teams (channels + provisioning)
  • Device Management: Intune (compliance + update rings)
  • Content: SharePoint Online documentation site
  • Automation: Microsoft Graph SDK + PowerShell 7
