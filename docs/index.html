Windows Server build guide (excerpt)

This guide walks through building a domain controller for the lab.local domain.

  1. Install Windows Server and apply updates.
  2. Assign a static IP and configure DNS to point to itself.
  3. Install the Active Directory Domain Services role.
  4. Promote the server to a new forest: lab.local.
  5. Verify replication, DNS, and basic authentication.
