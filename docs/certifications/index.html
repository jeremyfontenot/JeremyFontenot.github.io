Certifications

This page groups certification evidence that supports the technical work shown in projects and skills documentation. It helps recruiters quickly map credentials to practical capability in support, systems administration, networking, and cloud fundamentals. The linked detail page provides credential context while keeping the portfolio focused on applied outcomes.

Certifications are presented as supporting proof, not as a replacement for hands-on experience. They are paired with project and documentation evidence so reviewers can see both formal training and practical execution in the same portfolio flow.
