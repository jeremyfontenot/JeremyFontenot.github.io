Asset Inventory

The asset inventory records the systems, services, and supporting roles used in the documented home lab environment. It is written as a preserved operations reference, not as a claim of current production infrastructure. The inventory helps reviewers understand how virtualization, identity, network security, monitoring, and logging components relate to the project evidence shown elsewhere in the portfolio.

Each entry is intentionally high level so the page remains safe for public review while still showing practical infrastructure awareness. The role column explains why the asset matters operationally, and the status column identifies whether the item was part of the documented lab scope. Related architecture, runbook, validation, and troubleshooting pages provide additional context for implementation and review.
