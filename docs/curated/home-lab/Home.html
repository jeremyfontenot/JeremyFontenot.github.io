Overview

The Home Lab is a fully functional, cloud‑aligned environment designed to mirror real‑world enterprise infrastructure. It demonstrates identity, security, endpoint management, automation, and architecture skills using Microsoft 365, Entra ID, Intune, Windows Server, and PowerShell.

Objectives

  • Build a modern, cloud‑first environment aligned with Zero Trust principles.
  • Implement identity, device, and security baselines used in enterprise environments.
  • Create automation workflows for provisioning, reporting, and configuration.
  • Document architecture, decisions, and validation steps.
