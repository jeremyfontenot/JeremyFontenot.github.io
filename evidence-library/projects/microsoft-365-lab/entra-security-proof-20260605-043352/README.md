# Microsoft Entra Security Policy Proof Export

## Purpose

This folder contains read-only Microsoft Graph PowerShell exports for Entra security and identity policy review.

## What This Evidence Validates

This export can validate:

- Conditional Access policy inventory
- Conditional Access policy state
- Named location configuration
- Authentication methods policy
- Authentication method configuration inventory
- Authorization policy settings
- Security defaults enforcement policy state
- PowerShell transcript evidence of the read-only export session

- conditional-access-policies.json
- conditional-access-policies-summary.csv
- conditional-access-named-locations.json
- conditional-access-named-locations-summary.csv
- authentication-method-policy.json
- authentication-method-configurations.json
- authorization-policy.json
- security-defaults-policy.json
- entra-security-proof-summary.json

## Scope Note

This export is read-only. It supports portfolio proof of Microsoft Entra policy review and tenant security administration awareness. It does not claim production enterprise administration or client tenant ownership.
