# Microsoft Entra Security Proof Fallback Export

## Purpose

This folder contains supplemental Microsoft Graph request exports for Conditional Access and authentication method policy evidence.

## Evidence Value

This export records direct Microsoft Graph request results and captures any Graph service errors as preserved troubleshooting evidence.

## Files Created

- conditional-access-policies-graph-request.json
- conditional-access-policies-graph-request-beta.json
- authentication-method-policy-graph-request.json
- authentication-method-configurations-graph-request-beta.json
- graph-request-errors.csv
- entra-security-proof-fallback-summary.json
- entra-security-proof-fallback-transcript-20260605-043558.txt

## Scope Note

This is a read-only evidence capture. Failed Graph requests are preserved as troubleshooting evidence and should not be represented as successful policy exports.
