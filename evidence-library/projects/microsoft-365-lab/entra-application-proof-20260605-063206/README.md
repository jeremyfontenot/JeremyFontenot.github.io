# Microsoft Entra Application Governance Proof Export

## Purpose

This folder contains read-only Microsoft Graph PowerShell exports for Microsoft Entra app registrations, enterprise applications, service principals, and OAuth permission grants.

## What This Evidence Validates

This export can validate:

- Entra app registration inventory
- Enterprise application / service principal inventory
- Application sign-in audience metadata
- Publisher domain metadata
- OAuth permission grant review
- Application governance and consent-review awareness
- PowerShell transcript evidence of the read-only export session

## Files Created

- app-registrations.json
- app-registrations-summary.csv
- enterprise-applications-service-principals.json
- enterprise-applications-summary.csv
- oauth2-permission-grants.json
- oauth2-permission-grants-summary.csv
- application-export-errors.csv
- entra-application-proof-summary.json
- entra-application-proof-transcript-20260605-063206.txt

## Scope Note

This export is read-only. It supports portfolio proof of Microsoft Entra application governance review and cloud identity administration awareness.
