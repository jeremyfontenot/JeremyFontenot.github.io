# Artifacts

This folder holds local audit output, validation exports, and maintenance-time screenshots.

## Intended use

- `audits/` - generated site audit exports and review snapshots.
- `lighthouse/` - Lighthouse runs captured during validation.
- `screenshots/` - local screenshots for debugging and confirmation only.

## Rules

- Do not store production assets here.
- Do not promote temporary validation screenshots into the public site.
- Keep generated previews and scratch exports out of source-controlled content unless they are intentionally part of the published evidence library.
